# References

This folder holds general/cross-cutting research outputs and reference documents that do not belong to a single plan's task folder.

## What Goes Here

- Research outputs from RESEARCH mode that span multiple features
- External documentation snapshots, comparison studies, repo-pack artifacts
- Design references and source documents used across plans

## What Does Not Go Here

- References belonging to one plan (colocate inside that plan's task folder)
- Feature-specific references (use `process/features/{feature}/references/`)
- Durable project knowledge (belongs in `process/context/`)

## Naming Convention

`{topic}_{dd-mm-yy}.md` (or keep the original filename for imported artifacts)
